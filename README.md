## About

TypeScript and NestJS on Postgres, with Coolify, Kubernetes, and Terraform underneath — I like owning the full path from commit to on-call. React and Next.js when the product needs a web frontend; React Native (Expo) for mobile. Keycloak and Cerbos for identity and authorization. Redis shows up everywhere.

After hours: Rust, event sourcing, browser roguelikes, and retro-inspired operating systems.

## Featured

**[openobserve-mcp](https://github.com/kashik0i/openobserve-mcp)** — MCP server for multi-instance OpenObserve. Adaptive routing, request batching, LRU cache on the query path.
`typescript` · `mcp` · `observability`

**[coolify-mcp](https://github.com/kashik0i/coolify-mcp)** — 38 tools for driving Coolify from AI assistants. Curated by hand, not generated from the OpenAPI spec.
`typescript` · `mcp` · `paas`

**[beszel-mcp](https://github.com/kashik0i/beszel-mcp)** — Fork of the Beszel MCP with the upstream time-filter, container-stats schema, and bucket-sizing bugs fixed. Host and container metrics queryable from any MCP client.
`typescript` · `mcp` · `monitoring`

**[skull-empires-revived](https://github.com/kashik0i/skull-empires-revived)** — Browser director-style roguelike. Deterministic replay, SQLite-WASM saves over OPFS, procedural Web Audio score. Pure-TS core, thin adapters.
`typescript` · `vite` · `sqlite-wasm` · `web-audio`
